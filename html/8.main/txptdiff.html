txptdiff

(./main/txptdiff.m)

Function Synopsis

[t]=txptdiff(s,r,p,v)

Help text

 txptdiff: function to calculate the traveltime in a medium with constant 
 velocity v, from a point diffractor:
 INPUT: 
  s=[sx;sy;sz]  source position
  r=[rx;ry;rz]  receiver position 
  p=[x;y;z]  position of a POINT DIFFRACTOR  
 OUTPUT: travel time.
